How to Use This Admin Panel
  1. WhatsApp Orders: Update your WhatsApp number to receive orders directly from customers
  2. Logo Management: Upload your brand logo to replace the placeholder
  3. Product Management: Add new products with multiple images, colors, and sizes
  4. Categories: Organize products into T-Shirts, Shirts, Blouses, and Sweatshirts
  5. Image Upload: Upload high-quality product images for better customer experience
  6. Bulk Orders: Customers can specify bulk order requirements via WhatsApp
Note: This is a client-side admin panel. In a production environment, you would want to implement proper authentication and server-side storage for security.
